# Run Skip-Level And Manager System Review

Use this when managing managers, auditing manager effectiveness, investigating team-health gaps, onboarding first-time managers, or checking whether upward reports hide team reality.

1. Define scope and intent.
   - Verify: teams, managers, reason for review, confidentiality boundary, and expected output are explicit.
2. Choose skip-level format.
   - Verify: individual 1:1, group session, office hours, or observation is chosen based on org size and risk.
3. Prepare prompts.
   - Verify: questions cover team health, manager behavior, blockers, strategy clarity, collaboration, opportunities, and happiness/retention signal.
4. Collect signal.
   - Verify: notes separate facts, themes, direct quotes needing permission, and hypotheses.
5. Triangulate.
   - Verify: themes are checked against delivery data, incidents, attrition, calendars, tickets, manager 1:1s, and peer feedback.
6. Coach manager.
   - Verify: feedback is pattern-level, specific, non-punitive where appropriate, and tied to manager accountability.
7. Track follow-through.
   - Verify: themes, actions, owners, dates, and repeated unresolved signals are measured.

Output: skip-level signal map, manager accountability review, coaching plan, and quantified management-system risks.
